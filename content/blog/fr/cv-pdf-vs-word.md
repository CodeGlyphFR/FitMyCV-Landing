## Le format de votre CV peut tout changer

Vous avez peaufiné chaque ligne de votre CV. Les mots-clés sont en place, les expériences bien présentées, les compétences mises en valeur. Et pourtant, au moment de cliquer sur "Envoyer", une question surgit : PDF ou Word ?

Ce choix n'est pas anodin. Le format de fichier influence la manière dont votre CV est lu, affiché et analysé. Un mauvais choix peut compromettre des heures de travail, que ce soit à cause d'un ATS qui ne parvient pas à extraire vos données ou d'un recruteur qui voit une mise en page décalée.

## PDF : le format roi pour la mise en page

### Les atouts du PDF

Le PDF (Portable Document Format) a été concu pour que chaque lecteur voie exactement le meme document, quel que soit l'appareil ou le logiciel utilisé. Pour un CV, cela offre plusieurs avantages concrets :

- **Mise en page fidele :** colonnes, marges, espacements, polices sont identiques sur tout appareil.
- **Compatibilité universelle :** aucun logiciel payant requis pour lire un PDF.
- **Protection contre les modifications :** le recruteur ne peut pas altérer accidentellement votre contenu.
- **Poids léger :** un CV PDF pese rarement plus de 200 Ko.

### Les limites du PDF

- **PDF scannés :** si votre PDF est issu d'un scan (image), aucun ATS ne pourra en extraire le texte. Pour éviter ce piege, consultez [notre guide sur le format ATS](/blog/cv-compatible-ats-format).
- **Édition difficile :** une fois exporté, le PDF n'est pas simple à modifier sans outil dédié.
- **Certains anciens ATS :** une poignée de systemes tres anciens peinent encore avec les PDF, mais cela concerne moins de 5 % du marché en 2026.

### Le test de compatibilité

Pour vérifier que votre PDF est bien un PDF texte (et non une image), faites le test suivant :

1. Ouvrez votre fichier PDF
2. Ctrl+A (tout sélectionner)
3. Ctrl+C (copier)
4. Collez dans un éditeur de texte (Bloc-notes, TextEdit)

Si le texte apparait proprement et dans le bon ordre, votre PDF est compatible ATS. Si le résultat est vide, illisible ou dans le désordre, votre PDF pose probleme.

## Word (.docx) : le format historique du recrutement

### Les atouts du Word

Le format .docx reste le format natif de nombreux ATS, car il a longtemps été la norme dans le monde professionnel :

- **Parsing natif :** les ATS les plus anciens ont été construits pour lire du Word en priorité.
- **Édition facile :** vous pouvez modifier votre CV rapidement avant chaque candidature.
- **Métadonnées structurées :** Word offre une structure de document (titres, paragraphes, listes) que les ATS exploitent directement.

### Les limites du Word

- **Mise en page instable :** un fichier .docx ouvert sur LibreOffice, Google Docs ou une version différente de Word peut voir ses colonnes se décaler, ses polices changer, ses marges se réduire.
- **Polices non embarquées :** si le recruteur n'a pas la meme police que vous, le rendu change.
- **Risque de modification :** le recruteur peut accidentellement modifier votre contenu en ouvrant le fichier.
- **Macros et compatibilité :** les fichiers .doc (ancien format) ou les documents avec macros peuvent etre bloqués par les filtres de sécurité.

## Quand choisir le PDF

Le PDF est le choix par défaut dans la majorité des situations :

- **Candidature par email :** le recruteur verra exactement votre mise en page.
- **Dépot sur un site carriere :** la plupart des plateformes modernes acceptent et lisent le PDF sans probleme.
- **Offre sans précision de format :** en l'absence d'indication, le PDF est le choix le plus sur.
- **CV avec mise en page soignée :** si vous avez travaillé le design, le PDF préservera chaque détail.

## Quand choisir le Word

Le Word reste pertinent dans certains cas précis :

- **L'offre le demande explicitement :** certaines entreprises ou cabinets de recrutement demandent un .docx. Respectez toujours cette consigne.
- **Recrutement via agence :** les cabinets de recrutement reformatent souvent les CV avant de les transmettre au client. Un .docx leur facilite le travail.
- **ATS ancien identifié :** si vous savez que l'entreprise utilise un ATS plus ancien, le .docx réduit les risques de parsing incomplet.

## La stratégie optimale : préparer les deux

La meilleure approche consiste à préparer votre CV dans les deux formats :

1. **Rédigez dans Word** pour profiter de la structure native (titres, listes, styles).
2. **Exportez en PDF** via "Enregistrer sous > PDF" (et non pas via un outil d'impression virtuelle, qui peut générer un PDF image).
3. **Testez le PDF** avec le test Ctrl+A décrit plus haut.
4. **Conservez les deux versions** pour adapter votre envoi selon la demande.

Pour aller plus loin sur la structure et les polices à adopter, lisez [notre article dédié à la police et à la mise en page ATS](/blog/police-mise-en-page-cv-ats).

## Les erreurs fréquentes liées au format

### Envoyer un PDF image

Un CV scanné ou exporté depuis Canva sous forme d'image n'est pas lisible par les ATS. Le fichier semble normal visuellement, mais le systeme automatisé ne détecte aucun texte. C'est l'une des [erreurs les plus fréquentes](/blog/erreurs-cv-ats) chez les candidats.

### Utiliser un ancien format .doc

Le format .doc (Word 97-2003) est obsolete. Certains ATS le refusent, et les filtres anti-spam des entreprises le bloquent parfois. Utilisez toujours le .docx.

### Nommer le fichier de maniere générique

`CV.pdf`, `Document1.docx` ou `Mon CV final final (2).pdf` ne font pas bonne impression. Adoptez un nommage professionnel : `Prenom_Nom_CV.pdf`.

### Oublier de vérifier la compatibilité

Meme un PDF texte peut poser probleme si votre outil d'export a mal géré les polices ou les caracteres spéciaux. Vérifiez toujours le résultat final.

## L'impact du format sur votre score ATS

Le format de fichier n'affecte pas directement votre score de compatibilité avec une offre, car celui-ci repose sur l'analyse des mots-clés, des compétences et de l'expérience. En revanche, un format incompatible empeche l'ATS d'effectuer cette analyse, ce qui équivaut à un score de zéro.

Pour comprendre en détail comment le score de compatibilité est calculé, consultez [notre explication du score de compatibilité CV-offre](/blog/score-compatibilite-cv-offre).

## FAQ

**Peut-on envoyer un CV en PDF et en Word en meme temps ?**
Sauf indication contraire dans l'offre, envoyez un seul fichier. Privilégiez le PDF. Envoyer deux versions peut sembler hésitant et complique la gestion pour le recruteur.

**Google Docs est-il un bon format d'export ?**
Google Docs n'est pas un format de fichier, mais un outil. Vous pouvez exporter depuis Google Docs en PDF ou en .docx. L'export PDF de Google Docs produit un PDF texte compatible ATS dans la grande majorité des cas.

**Les ATS modernes ont-ils encore des problemes avec le PDF ?**
En 2026, la quasi-totalité des ATS du marché (Workday, Greenhouse, Lever, Taleo, SmartRecruiters) lisent le PDF texte sans difficulté. Les problemes se limitent aux PDF images ou aux PDF avec des mises en page tres complexes (colonnes multiples imbriquées, zones de texte flottantes).

**Et le format Pages d'Apple ?**
Ne l'envoyez jamais tel quel. Les ATS et les recruteurs sous Windows ne peuvent pas l'ouvrir. Exportez en PDF depuis Pages.
